# 나의 입시멘토 안내문

## 1. 서비스명

```text
나의 입시멘토
탐구·세특·입시위치진단
```

- 위 2줄은 반드시 한꺼번에 한 세트로 사용합니다
- 한 줄만 따로 떼어 표기하지 않습니다
- 두 줄 사이에 다른 문구를 넣지 않습니다

## 2. 서비스 소개

나의 입시멘토는 학생 정보, 학생부 분석, 탐구활동/독서/세특 제안, 입시위치 진단을 하나의 흐름으로 연결하는 4단계 서비스다.

## 3. 이용 흐름

### 1단계 학생정보입력

- 사용자정보 입력과 학생부를 등록/진단합니다
- 학생 기본 정보와 연락처를 입력합니다
- 학생부 PDF를 등록해 성적 요약을 반영할 수 있습니다

### 2단계 학생부 분석

- 학생부 상세 분석결과를 리포트로 제공합니다
- 학생부 원문, 요약 비교 분석, 비교 평가를 한 화면에서 확인합니다
- 인쇄와 메일 보내기를 제공합니다

### 3단계 탐구활동/독서/세특

- 학생부 추출 주제 및 개별 주제로 탐구활동, 독서, 세특문장을 제안합니다
- 기본 3개 주제가 먼저 제공됩니다
- 추가 검색으로 최대 3개까지 확장할 수 있습니다
- 전체 결과를 한 번에 인쇄하거나 메일로 보낼 수 있습니다

### 4단계 입시위치 진단

- 내신성적과 학생부 기반으로 희망대학매칭 진단 및 결과를 제시합니다
- 사용자가 직접 선택한 대학 기준으로 판정 결과를 확인합니다
- 판정은 `하향 / 안정 / 도전 / 불가` 로 표시됩니다
- 최종 결과를 인쇄하거나 메일로 보낼 수 있습니다

## 4. 결과 확인 방식

- 단계별 결과는 각 화면 안에서 바로 확인합니다
- 메일 입력창은 별도 페이지가 아니라 해당 화면 내부에 열립니다
- 각 단계 결과는 개별 보고서 단위로 발송됩니다

## 5. 주의 사항

- 서비스명은 항상 2줄 한 세트, 좌측 정렬 기준으로 사용합니다
- `AI 탐구세특 생성기`, `우리아이 입시위치진단` 명칭은 사용하지 않습니다
- 단계명과 단계 설명 문구는 고정값으로 유지합니다
