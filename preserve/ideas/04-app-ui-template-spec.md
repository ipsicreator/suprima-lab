# 나의 입시멘토 UI 기준서

## 1. 서비스명

```text
나의 입시멘토
탐구·세특·입시위치진단
```

- 서비스명은 모든 화면에서 2줄 유지
- 아래 2줄은 반드시 한꺼번에 한 세트로 사용
- 1줄 `나의 입시멘토`는 작게 표시
- 2줄 `탐구·세특·입시위치진단`은 더 크게 표시
- 두 줄은 항상 좌측 정렬
- 두 줄 중 한 줄만 단독 사용하지 않음
- 두 줄 사이를 다른 문구로 끊지 않음
- 두 줄을 가로로 나열하거나 가운데 정렬하지 않음

## 2. 공통 셸 기준

- 상단 좌측: 대치 수프리마 로고
- 상단 중앙: 서비스명 2줄
- 상단 우측: `PREMIUM DIAGNOSIS`
- 단계 바: 1~4단계 공통 진행바
- 현재 단계: 버건디 원형 강조
- 본문 상단: 단계 제목
- 본문 설명: 단계별 확정 문구 사용
- 결과 화면: 빈 안내 카드가 아니라 실제 입력값, 분석값, 결과값 표시

## 3. 단계명 및 설명

### 1단계: 학생정보입력

```text
사용자정보 입력과 학생부를 등록/진단합니다
```

### 2단계: 학생부 분석

```text
학생부 상세 분석결과를 리포트로 제공합니다
```

### 3단계: 탐구활동/독서/세특

```text
학생부 추출 주제 및 개별 주제로 탐구활동, 독서, 세특문장을 제안합니다
```

### 4단계: 입시위치 진단

```text
내신성적과 학생부 기반으로 희망대학매칭 진단 및 결과를 제시합니다
```

## 4. 단계별 화면 기준

### 1단계 학생정보입력

- 입력 목적은 사용자 정보와 학생부 PDF 등록
- 입력 필드:
  - 컨설턴트명
  - 학생 이름
  - 학교명
  - 학년
  - 학생 연락처
  - 학부모 연락처
  - 이메일
  - 희망 진로/학과
  - 등급 체계
  - 내신 등급
- PDF 업로드 영역 유지
- PDF 분석 후에는 성적 요약 배지까지 노출
- 학생부 분석 프리뷰 카드는 1단계에서 노출하지 않음

### 2단계 학생부 분석

- 참조 구조는 `1분면 / 2분면 / 3분면`
- 상단 안내 영역:
  - 학생부종합전형 평가 가상 화면 제목
  - 학생부 분석용 시뮬레이션 안내 문구
  - 지원자 정보 칩
  - 체험판 또는 로컬 저장 상태
- 1분면:
  - 학생부 원문
  - 키워드 검색
  - 모집단위 전환
  - 학생부 매핑
  - 학생부 원문 섹션 카드
- 2분면:
  - 요약·비교 분석
  - 학기별 등급 추이
  - 성취도 레이더
  - 핵심 키워드
- 3분면:
  - 비교·평가
  - 선택 과목 조합 평균 추이
  - 성취도 분포
  - 입학사정관 평가 메모
- 하단 액션:
  - 인쇄
  - 메일 보내기

### 3단계 탐구활동/독서/세특

- 기본 3개 주제를 먼저 노출
- 각 주제 구성:
  - 탐구 주제
  - 논문
  - 자료
  - 독서
  - 세특문장
- 개인 주제 추가 검색:
  - 과목 선택
  - 추가 주제 입력
  - 진로/학과 힌트
  - 검색 버튼
- 추가 주제는 최대 3개
- 전체 주제 수는 총 3~6개
- 인쇄/메일 보내기는 전체 주제 묶음 기준 1회

### 4단계 입시위치 진단

- 최종 제목은 `입시위치 진단`
- 상단 결과 요약:
  - 희망대학 n개 진단
  - `하향 · 안정 · 도전 · 불가` 합산 노출
- 전 단계 요약:
  - 1단계 학생정보입력
  - 2단계 학생부 분석
  - 3단계 탐구활동/독서/세특
- 대학 결과:
  - 기본 3개 대학 아코디언
  - 4개째부터 추가 버튼으로 확장
  - 필드: 대학, 모집단위, 전형유형, 전형명, 판정, 진단근거
- 판정값:
  - 하향
  - 안정
  - 도전
  - 불가
- 하단 액션:
  - 인쇄
  - 메일 보내기

## 5. 메일/출력 기준

- 2단계: 학생부 분석 결과만 발송
- 3단계: 탐구활동/독서/세특 결과만 발송
- 4단계: 입시위치 진단 결과만 발송
- 각 단계는 개별 인쇄 가능
- 메일 입력 UI는 각 화면 내부에서 확장 표시

## 6. 금지 사항

- 단계명 임의 변경 금지
- 단계 설명 문구 임의 변경 금지
- `AI 탐구세특 생성기` 명칭 사용 금지
- `우리아이 입시위치진단` 명칭 사용 금지
- 서비스명 구분기호를 공백이나 `+` 로 대체 금지
- `나의 입시멘토` 와 `탐구·세특·입시위치진단` 을 따로 떨어뜨려 사용 금지
- 서비스명 2줄 분리 배치 금지
- 이미지 시안과 실제 구현 상태 혼동 금지
- 앱 구동에 필요한 파일 삭제 금지
