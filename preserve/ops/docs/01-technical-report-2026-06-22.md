# 나의 입시멘토 기술보고서

기준일: 2026-06-22

## 1. 개요

오늘 작업 기준 서비스는 `나의 입시멘토`이며, 상단 브랜드 표기는 아래 2줄로 고정했다.

```text
나의 입시멘토
탐구·세특·입시위치진단
```

위 2줄은 반드시 함께 묶어서 사용하며, 한 줄만 단독 사용하지 않는다.

이번 작업의 목적은 1~4단계 진단 흐름을 공통 템플릿 안에서 정렬하고, 각 단계가 실제 입력값과 결과값을 보여주는 상태로 맞추는 것이다.

## 2. 반영 범위

- 1단계 학생정보입력 정리
- 2단계 학생부 분석 대시보드 구조 반영
- 3단계 탐구활동/독서/세특 결과 화면 반영
- 4단계 입시위치 진단 결과 화면 반영
- 공통 상단 브랜드/단계 바 반영
- 단계별 인쇄/메일 동작 유지
- 최신 화면 캡처 재생성

## 3. 구현 상세

### 3.1 공통 셸

- 좌측 로고, 중앙 서비스명 2줄, 우측 `PREMIUM DIAGNOSIS` 배치
- 1~4단계 공통 진행 바 구성
- 현재 단계 강조 처리
- 모든 단계에 동일한 타이포 구조 적용

### 3.2 1단계

- 사용자 입력 필드 구성 완료
- 학생부 PDF 업로드 영역 유지
- PDF 분석 결과는 성적 요약 수준만 노출
- 학생부 분석 프리뷰 카드는 제거

### 3.3 2단계

- 1분면 학생부 원문
- 2분면 요약·비교 분석
- 3분면 비교·평가
- 차트와 분석 카드, 키워드 표시, 메일/인쇄 버튼 유지

### 3.4 3단계

- 기본 3주제 선노출
- 주제별 탐구 주제, 논문, 자료, 독서, 세특문장 구성
- 개인 주제 추가 검색으로 최대 3개 확장
- 전체 3~6개 결과를 1회 인쇄/메일 발송

### 3.5 4단계

- 전 단계 요약 카드 배치
- 기본 3개 희망대학 아코디언 구성
- 대학별 판정 `하향 / 안정 / 도전 / 불가`
- 대학 추가 버튼 유지
- 인쇄/메일 버튼 유지
